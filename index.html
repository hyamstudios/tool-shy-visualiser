Add soft hyphens

Soft hyphens are a way to indicate to browsers where words should break if it needs to be broken accross multiple lines without adding visible hyphenation. The problem with soft hyphens is that they are invisible characters which makes them difficult to use. This tool will visualise soft hyphens as you add them to words.

To learn more check out the soft hyphen wikipedia entry

Soft hyphens are invisible characters are invisible which makes them hard to copy and past. To help you can use this button to copy one to your clipboard. You can then use the input field below to type the word you want to hyphenate and paste the soft hyphen into. The hypen will then be visualised in the output area. This output can then be copied and used where needed.
